Notions#

Les notions de vecteur, de normes et de coordonnées de vecteur doivent être maitrisé.

Changement de repère#

Il existe plusieurs repères sur le terrain :

  • repère t - terrain (axe u⃗,v⃗\vec{u},\vec{v}u,v)
  • repère r - robot (axe tangent⃗,normale⃗\vec{tangent},\vec{normale}tangent​,normale)

Notons les éléments suivants :

  • tv⃗{}^t\vec{v}tv : Coordonées du vecteur vitesse v⃗\vec{v}v exprimé dans le repère ttt.
  • rv⃗{}^r\vec{v}rv : Coordonées du vecteur vitesse v⃗\vec{v}v exprimé dans le repère rrr.
  • rRt{}^rR_trRt​ : Matrice de rotation dans le repère RRR au repère TTT.
  • θ\thetaθ : orientation du robot
